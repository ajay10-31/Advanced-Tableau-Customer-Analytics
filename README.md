# Advanced Tableau Customer Analytics

## Business Problem
A subscription-based company wants to understand customer lifetime value (CLV), churn risk, and revenue exposure in order to make informed retention and revenue decisions.

## Key Business Questions
- Who are the most valuable customers over their lifetime?
- Which customers are at risk of churn?
- How much revenue is currently at risk?
- Which customer segments should be prioritized for retention efforts?

## Project Scope
This project combines business analytics and data analysis using Tableau to model customer behavior, profitability, and churn risk using transactional, engagement, and cost data.

## Tools & Technologies
- Tableau Desktop
- CSV datasets
- Git & GitHub
